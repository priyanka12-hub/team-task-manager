# Team Task Manager

This is a simple Team Task Management web application built using HTML, CSS, and JavaScript.

## Features
- Create projects
- Add tasks with due dates
- Track task status (To Do)
- View total, completed, and pending tasks

## Tech Stack
- HTML
- CSS
- JavaScript
